# Phase 1 – UI Proof

## 목적
Lemon Squeezy 실결제가 가능한 거래 흐름을 "눈으로 증명"하기 위한 최소 UI

## 실행

```bash
npm run dev
```

## 테스트 시나리오

1. http://localhost:3000 접속
2. "Create Deal" 버튼 클릭
3. "🚗 Used Car Transaction" 버튼 클릭
4. Deal 상세 페이지에서 "Proceed Payment" 버튼 확인

## 성공 조건

- ✅ Landing 페이지 표시
- ✅ Deal 생성 흐름 동작
- ✅ 결제 버튼 표시
- ✅ Escrow 설명 표시
